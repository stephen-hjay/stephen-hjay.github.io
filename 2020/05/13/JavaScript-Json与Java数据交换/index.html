JavaScript|Json与Java数据交换

Posted on 2020-05-13

JSON：JavaScript Object Notation

概念： JavaScript Object Notation JavaScript对象表示法

​ var p = {“name”:”张三”,”age”:23,”gender”:”男”};

  • json现在多用于存储和交换文本信息的语法

进行数据的传输

  • JSON 比 XML 更小、更快，更易解析。

语法：

  • 数据在名称/值对中：json数据是由键值对构成的
  • 键用引号(单双都行)引起来，也可以不使用引号
  • 值得取值类型：
    1. 数字（整数或浮点数）
    2. 字符串（在双引号中）
    3. 逻辑值（true 或 false）
    4. 数组（在方括号中） {“persons”:[{},{}]}
    5. 对象（在花括号中） {“address”:{“province”：”陕西”….}} json中的json对象
    6. null
  • 数据由逗号分隔：多个键值对由逗号分隔
  • 花括号保存对象：使用{}定义json 格式。就是 Json本质上就是一种对象字典
  • 方括号保存数组：[]

获取数据:

  1. json对象.键名（注意是直接打键名）
  2. json对象[“键名”] 适用于遍历键名
  3. 数组对象[索引]

JSON数据和Java对象的相互转换

服务器需要将json的数据 转换为Java对象进行利用。

服务器从客户端（浏览器）获取json数据，并将其转换为Java对象进行数据获取，服务器将计算得到的Java对象 转换为json数据 返回给浏览器端进行解析。

  • JSON解析器：
    • 常见的解析器：Jsonlib（官方提供的），Gson（Google提供的），fastjson（阿里巴巴提供的），jackson（springMVC 内置解析器）

本质就是javascript字典与JavaBean (JavaBean数组 或者是Map) 之间的转换

JSON转为Java对象

  1. 导入jackson的相关jar包
  2. 创建Jackson核心对象 ObjectMapper
  3. 调用ObjectMapper的相关方法进行转换
    1. readValue(json字符串数据, JavaBean.Class)

Java对象转换JSON

  1. 导入jackson的相关jar包

  2. 创建Jackson核心对象 ObjectMapper

  3. 调用ObjectMapper的相关方法进行转换

    有2种主要的方法：

    • writeValue(参数1，obj):
      File：将obj对象转换为JSON字符串，并保存到指定的文件中
      Writer：将obj对象转换为JSON字符串，并将json数据填充到字符输出流中
      OutputStream：将obj对象转换为JSON字符串，并将json数据填充到字节输出流中
    • writeValueAsString(obj):将对象转为json字符串
2. 注解：加在JavaBean的字段上
 3. @JsonIgnore：排除属性。
 4. @JsonFormat：属性值得格式化

  • @JsonFormat(pattern = “yyyy-MM-dd”)

    1. 复杂java对象转换
    2. List：数组 可以直接转换
    1. Map：对象格式一致 就是一个JavaBean
